## âš™ï¸ Installation & Setup

This project requires **3 Terminal Windows** running simultaneously.

### 1. Prerequisites

- Node.js & npm
- Python 3.9+
- MetaMask Browser Extension
- (Optional) Ganache UI

### 2. Blockchain Setup (Terminal 1)

Deploy the Smart Contracts to your local network.

```bash
# Install dependencies
npm install

# Compile Contracts
npx hardhat compile

# Deploy Economy System (Token -> Weapon -> Monster)
npx hardhat run scripts/deploy-economy.js --network ganache

```

_Copy the `WEAPON`, `MONSTER`, and `TOKEN` addresses outputted here for Step 4._

### 3. AI Backend Setup (Terminal 2)

Launch the Python server that talks to Google Gemini.

```bash
cd backend

# Install Python requirements
pip install fastapi uvicorn langchain langchain-google-genai pydantic

# Set your API Key (Or set it in main.py)
export GOOGLE_API_KEY="your_api_key_here"

# Start Server
uvicorn main:app --reload

```

### 4. Frontend Setup (Terminal 3)

Launch the React Interface.

```bash
# Go back to root if needed
npm install

# Start React
npm run dev

```

**Configuration:** Open `src/App.jsx` and replace the placeholder addresses with the ones generated in Terminal 1.

## ðŸŽ® How to Play

1. **Connect Wallet:** Click "Connect" to link MetaMask (Ensure you are on Localhost:8545).
2. **The Forge:** Type a prompt (e.g., _"A cybernetic hammer made of neon glass"_).
3. **Mint:** The AI generates the stats and art. Sign the transaction to mint it as an NFT.
4. **The Arena:** Equip your weapon from the "Armory" and attack the Monster.
5. **Earn:** Killing the monster rewards you with **$GOLD**.
6. **Upgrade:** Use your `$GOLD` balance to upgrade your weapon (Level +1), increasing its damage for future fights.

## ðŸ“œ Smart Contract Architecture

- **`AIWeapon.sol` (ERC-721):** Stores the NFT data. Contains `upgradeWeapon()` function which burns tokens to increase level.
- **`GameToken.sol` (ERC-20):** The currency. Only the Monster contract is authorized to mint new tokens.
- **`Monster.sol`:** Game engine. Verifies weapon ownership via `Interface`, calculates damage, and distributes rewards.
